# Contributing

This project is published as a portfolio / reference implementation. It is **not open to external contributions** — pull requests will not be reviewed or merged.

If you want to use, study, or adapt the code for your own work: **fork it freely** and modify your fork. The code is MIT-licensed (see `LICENSE`) — you can do essentially anything you like with your copy, including running it commercially, with no obligation to upstream.

If you've found a real bug that affects others, you're welcome to email the author at the address listed in `pyproject.toml`. Response is best-effort.

Thanks for your interest.
